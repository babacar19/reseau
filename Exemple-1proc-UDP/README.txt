exempleLOCAL :
cree les prises Emission/Reception (primitives UDP)
sur la boucle locale 
puis envoie une chaine de caracteres (sans espace), 
la recoit et l'affiche sur la sortie standard

Compilation :
	make -f Makefile-exempleLOCAL [cible]

Usage : 
	./exempleLOCAL hello
	=> emission/reception de "hello"
